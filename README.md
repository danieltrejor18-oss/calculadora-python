# Calculadora Simple en Python

Calculadora de consola que realiza operaciones matemáticas básicas.

## Características
- Suma, resta, multiplicación, división y porcentaje
- Manejo de errores para entradas inválidas
- Manejo de división entre cero
- Funciona en bucle hasta que el usuario decida salir

## Cómo usarla
1. Ejecutar el archivo calculadora.py
2. Ingresar dos números
3. Elegir la operación (suma/resta/division/multiplicar/porcentaje)
4. Escribir "salir" cuando quieras terminar

## Tecnologías
- Python 3
